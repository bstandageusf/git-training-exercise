# Git Training Exercise

This repository is used for hands-on Git training. Each participant will complete small tasks that demonstrate core Git workflows such as branching, committing, merging, and resolving conflicts.

## Goals

- Practice creating and switching branches  
- Make and commit changes  
- Push and pull from a remote  
- Resolve merge conflicts  
- Review and merge pull requests

## Instructions

1. Clone the repository.  
2. Create a personal branch using your name/team name.  
3. Complete the assigned exercises in the `exercisess.md` file.  
4. Commit and push your changes.
5. Merge the most up to date version of main into your branch
6. Resolve conflicts if they occur and complete the merge.
7. Open a pull request to merge your branch.  

## Notes

This repo is a safe place to experiment. Break things, fix them, and learn in the process.
